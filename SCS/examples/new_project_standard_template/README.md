# New Project Standard Template

这是一个用于从零启动正式长篇的新项目标准示例目录。

## 推荐栈
- V1.0 Core
- V1.1a Route Layer
- V1.1b Workbench Layer

## 使用顺序
1. 填写 `project_bible.example.json`
2. 填写 `canon_rules.example.json`
3. 填写 `character_state.example.json`
4. 填写 `volume_plan.example.json`
5. 创建 `chapter_001_contract.example.json`
6. 生成 `chapter_001_run_packet.example.json`
7. 按章节闭环推进

## 适合
- 从零开正式长篇
- 题材明确
- 不想一上来就全开最重治理层

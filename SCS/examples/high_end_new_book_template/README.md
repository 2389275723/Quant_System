# High-End New Book Template

这是给 **最高级新书栈** 准备的示例目录。

## 推荐栈
- V1.0 Core
- V1.1a Route Layer
- V1.1b Workbench Layer
- V1.1c Deep Governance Layer
- V1.2b Auto Governance Layer

## 适合
- 多卷长篇
- mixed 项目
- 世界层级高、关系复杂、终局明确的项目
- 想从一开始就把新书当系统工程管理的项目

## 推荐使用顺序
1. 填写 `project_bible.example.json`
2. 填写 `canon_rules.example.json`
3. 填写 `character_state.example.json`
4. 填写 `volume_plan.example.json`
5. 填写 `project_status_dashboard.example.json`
6. 填写 `chapter_001_contract.example.json`
7. 填写 `chapter_001_run_packet.example.json`
8. 初始化 `milestone_tracker.example.json`
9. 初始化 `character_drift_monitor.example.json`
10. 初始化 `governance_priority_panel.example.json`

## 额外建议
- 正文写作模型和治理模型最好分开
- 新书默认不带 V1.2a Legacy & Migration
- 先锁 pack，再锁 foundation，再跑 chapter flow

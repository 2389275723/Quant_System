# SCS 当前推荐实战包

## 1. 新项目标准包

适合：从零开正式长篇、20章以上、题材明确。

推荐栈：

- V1.0 Core
- V1.1a Route Layer
- V1.1b Workbench Layer

## 2. 复杂新项目包

适合：50章以上、mixed 项目、多卷、多层世界、多主轴。

推荐栈：

- V1.0 Core
- V1.1a Route Layer
- V1.1b Workbench Layer
- V1.1c Deep Governance Layer

## 3. 旧项目纳管包

适合：已写旧项目、半途项目、旧坑修复。

推荐栈：

- V1.0 Core
- V1.2a Legacy & Migration Layer
- V1.1b Workbench Layer

## 4. 完整治理包

适合：大型项目、复杂 mixed 项目、长期工程化治理。

推荐栈：

- V1.0 Core
- V1.1a Route Layer
- V1.1b Workbench Layer
- V1.1c Deep Governance Layer
- V1.2a Legacy & Migration Layer
- V1.2b Auto Governance Layer

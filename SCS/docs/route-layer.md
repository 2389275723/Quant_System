# V1.1a Route Layer

## 已支持主路由

- `general`
- `xianxia`
- `high_concept_relationship`
- `court_intrigue`
- `western_fantasy`

## 核心原则

1. 先判主轴，再判副轴。
2. primary_route 决定根骨。
3. secondary_routes 不可被忽略，只要影响稳定性就必须补最小关键模块。
4. 不允许用“先写几章再补”绕过题材前置层。

## 当前主要用途

- 修仙长篇
- 强关系长篇
- 古代权谋长篇
- 西幻 / 史诗奇幻长篇

## 输出目标

统一输出：

- primary_route
- secondary_routes
- required_bootstrap_packs
- required_modules
- route_status
- write_permission

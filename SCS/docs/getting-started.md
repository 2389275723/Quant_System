# SCS 快速开始

## 1. 先选实战包

### 新项目标准包
- `V1.0 + V1.1a + V1.1b`

### 复杂新项目包
- `V1.0 + V1.1a + V1.1b + V1.1c`

### 旧项目纳管包
- `V1.0 + V1.2a + V1.1b`

### 完整治理包
- `V1.0 + V1.1a + V1.1b + V1.1c + V1.2a + V1.2b`

## 2. 新项目最小启动顺序

1. `project_init_wizard`
2. `bootstrap_router`
3. 进入对应题材 bootstrap pack
4. 锁定 `project_bible / canon_rules / character_state / volume_plan`
5. 建立第 1 章 `chapter_contract`
6. 生成 `chapter_run_packet`
7. 开始单章闭环

## 3. 旧项目接入顺序

1. `legacy_project_intake`
2. `legacy_route_reconstructor`
3. `foundation_backfill_builder`
4. `chapter_history_reconstructor`
5. `legacy_issue_scan`
6. `legacy_payoff_recovery_map`
7. `legacy_finale_recovery_map`
8. `migration_mode_selector`
9. `legacy_reentry_gate`

## 4. 当前默认建议

如果你要从零开一个正式长篇，默认先用：

**V1.0 + V1.1a + V1.1b**

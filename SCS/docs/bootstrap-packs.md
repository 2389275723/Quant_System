# Bootstrap Packs

当前仓库已为以下题材预留 bootstrap pack 模板目录：

- `templates/bootstrap/xianxia/`
- `templates/bootstrap/high_concept_relationship/`
- `templates/bootstrap/court_intrigue/`
- `templates/bootstrap/western_fantasy/`

建议用法：

1. 先经过 `bootstrap_router`
2. 进入对应 pack
3. 先锁 pack 核心模块
4. 再进入 foundation 层

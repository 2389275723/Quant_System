# V1.1c Deep Governance Layer

## 四个核心治理器

- `character_drift_monitor`
- `world_expansion_auditor`
- `payoff_fatigue_monitor`
- `premature_reveal_guard`

## 解决的问题

- 主角或配角慢性漂移
- 世界层级和设定慢性膨胀
- payoff / 爽点 / 情绪回报慢性疲劳
- 终局资产、世界真相、身份底牌被提前透支

## 推荐触发节奏

- 每 10 章：角色漂移 / payoff 疲劳
- 每卷结束：世界膨胀审计
- 卷高潮前：提前泄露防护
- 最终卷前：四项全检

# SCS Start Here

如果你第一次进入这个仓库，按下面顺序看：

## 1. 先判断你属于哪种使用场景

### 从零开新项目
先看：
- `docs/getting-started.md`
- `docs/recommended-stacks.md`
- `presets/new_project_standard.json`

### 开复杂长篇 / mixed 项目
先看：
- `docs/recommended-stacks.md`
- `docs/workbench-layer.md`
- `docs/deep-governance.md`
- `presets/new_project_complex.json`

### 接管旧项目 / 老坑 / 半途项目
先看：
- `docs/legacy-migration.md`
- `presets/legacy_project_migration.json`

### 想开完整治理形态
先看：
- `docs/auto-governance.md`
- `docs/auto-governance-runtime.md`
- `presets/full_governance.json`

## 2. 再看核心总图
- `README.md`
- `INDEX.md`
- `docs/architecture.md`
- `docs/repo-structure.md`
